## Getting started

```bash
# clone the project
$ git clone https://github.com/jiangzhangxiang/blog.git
# enter the project directory
$ cd blog
# install dependency
$ yarn install
# develop
$ yarn dev
```

This will automatically open http://localhost:8080

## Build

```bash
# build for test environment
$ yarn run build
```
