    # 两种盒子模型（box-sizing）的区别

    第一种盒模型是 content-box，即width制定的是 content 区域的快读，而不是实际宽度

    实际宽度 = width + padding + border

    第二种盒模型是 border-box，即 width 指定的是左右边框外侧的距离

    实际快读 = width
